# Azure Cosmos DB – DP-900 Lab Project

A hands-on Azure Cosmos DB project completed as part of the Microsoft Learn **DP-900: Azure Data Fundamentals** course.  
This project demonstrates core NoSQL concepts, JSON document modeling, and SQL API querying within the Azure Portal.

## 📘 Overview
This project documents my work completing the Azure Cosmos DB lab from the DP-900 learning path.  
The lab focuses on creating a Cosmos DB resource, inserting JSON documents, defining partition keys, and running SQL API queries to explore how NoSQL databases operate in Azure.

## 🎯 Objectives
- Create and configure an Azure Cosmos DB account  
- Understand databases, containers, and partition keys  
- Insert and manage JSON documents  
- Run SQL API queries to filter and retrieve data  
- Explore NoSQL concepts and Azure data services  

## 🛠️ What I Did
- Provisioned an Azure Cosmos DB account in the Azure Portal  
- Created a database and container with a defined partition key  
- Added items (documents) to the container  
- Executed SQL API queries to retrieve and filter data  
- Explored the Azure Portal interface for Cosmos DB  

## 🧰 Technologies Used
- Azure Cosmos DB (SQL API)  
- Azure Portal  
- JSON  
- NoSQL data modeling
